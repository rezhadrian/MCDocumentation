Detailed Description

template<typename _Ty>
struct RawData< _Ty >

RawData structure handles a field data, which is set in the form of standard contaier, i.e. std::vector. This structure inherits from __mc_data__ ["Data"] base class using _Ty floating-point type and __mc_data_initializer__ [DataInitializer<std::vector<_Ty>, GridSizes>] initializer type.

template<typename _Ty >
MCbool RawData< _Ty >::initialize ( const Initializer &  init)
inlineoverridenoexcept

Initializes an instance with data read from the elements of custom Initializer tuple. One can also use this call to set up the reporter api::MetaCall to be able to screen-print a meta information about user-defined data structures.

This structure extracts a "raw data" and grid sizes from the tuple init and passes them as a closure to the handler target. Values are mapped accordingly using __mc_zorder_to_pos__ ["<tt>core::posToZorder(const GridSizes, const GridSizes) -> MCindex</tt>"].

Returns
True, if the initialization executed successfully. It is recommended to always check the return value, although it is not necessary.
