This is an API documentation for MC. For first time users, it is recommended to check the repository and tutorial linked the bottom of this page. For a deeper look into the algorithm and the maths behind it, make sure to visit the reference page.

MC is a fast and convenient toolkit to process different geometry configurations based on the classic Marching Cubes algorithm. The library utilizes modern C++, takes advantages of multi-core CPU, and focuses on modern API, usability, and ease of integration with other projects.

The library is developed by Maximenko, I., Tang, Q., Tanuharja, R.A., Oztoprak, O. (supervisor).
