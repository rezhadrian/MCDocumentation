Detailed Description

template<typename _Ty>
class RegularMesh< _Ty >

RegularMesh class is used for an output creation.

This type of mesh disrespects the uniqueness of the triangulated points, however, allows much more flexible mastering of parallelization process. When being appended all points are enumerated in the increasing order as they come.
