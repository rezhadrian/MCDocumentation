Detailed Description

core namespace contains various auxiliary classes, constants and methods, which are used during computational pipeline. Computational core of the marching cubes algorithm.

◆ zorderToPos()

)
inlinenoexcept

Unlike __mc_zorder_to_pos__ ["<tt>core::posToZorder(const GridSizes, const GridSizes) -> MCindex</tt>"] this function goes exactly the other way around: from the linear index under the given grid sizes it computes 3-index of the matching point. If at least one of the sizes is equal to zero, {-1, -1, -1} array is returned.
