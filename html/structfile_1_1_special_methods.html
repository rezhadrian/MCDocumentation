Splits string into substrings by the given separator.

Parameters
stringA target String.
separatorA given separation String;
Returns
A __mc_vector_string__ ["vector of strings"], formed out of the original string out of the split procedure.
Note
There are 2 special scenarios of the split call. If the length of separator is greater or equal to the length of string, then the output vector consists only of string. If the separator is an empty string, then string is parced into MCchar sequence and copied elementwise to the output vector.
